# 🏠 ResolveX – AI-Powered Hostel Complaint & Room Allocation Management System
> An intelligent hostel management system that automates complaint handling and room allocation using Machine Learning for complaint prioritization.

# 📌 Overview
ResolveX is an AI-powered hostel management web application designed to streamline complaint management and room allocation within educational institutions. The system enables students to raise complaints, monitor their status, and request room allocations, while providing administrators with a centralized dashboard for efficient hostel management.

Unlike traditional complaint systems, ResolveX incorporates Natural Language Processing (NLP) to automatically analyze complaint descriptions and assign priority levels (High, Medium, or Low), enabling administrators to address critical issues first.


# ✨ Features
## Student Module

- Student Registration & Login
- Secure Authentication
- Raise Hostel Complaints
- Complaint History
- Complaint Status Tracking
- Upload Complaint Images
- Room Allocation Request
- View Room Details
- Receive Admin Updates

## Admin Module

- Secure Admin Login
- Student Management
- Complaint Management
- Room Management
- Approve / Reject Room Requests
- Complaint Analytics Dashboard
- Search & Filter Complaints
- Update Complaint Status

## AI Module

- Complaint Priority Prediction
- TF-IDF Feature Extraction
- Linear Support Vector Machine (Linear SVM)
- Automatic High / Medium / Low Classification
- Machine Learning Model Evaluation


# 🧠 AI Complaint Prioritization
The AI module analyzes complaint descriptions and predicts their urgency using Natural Language Processing.
